## Short Explanation

<br />

Prime Partner is an online dating and geosocial networking application. In Prime, users can like or dislike other users' profiles, which include their photos, a short bio, and a list of their interests.

<TechStack data={['vite', 'reactjs', 'zustant', 'reactQuery']} />

- Zustant: manage global states and handle authenticate user
- Vite: build and config source code
- Mantine: lib ui for build components faster and practice with it
- React Query + Axios: fetch data from api and handle response with cache
